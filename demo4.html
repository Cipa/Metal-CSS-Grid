Grid in Grid - experimental feature

With a few lines of css you can add support for using the grid inside the main grid.

In order to make this work I introduced a new class name: .inGrid. Just add this class to the grid inside the main grid and the left padding for the first column and right padding for the last column will be removed.

With a few lines of code you can control the way the secondary grid is transformed at different screen sizes. Have a look at demo4.css
